Comparison Operators in Python

Python supports the standard comparison operators: <, >, ==, and !=. Each of these returns a boolean value.

Examples with Numbers

Here are some examples of comparisons with numeric values:

print("Less than") print(1 < 2) print(1 < 1) print(2 < 1) print("Greater than") a = 1 b = 2 print(a > 2) print(1 > a) print(b > a) print("Equality") a = 1 b = 2 print(a == 1) print(a == b) print("Inequality") print(a != 1) print(a != b) print("Less than or equal, Greater than or equal") print(5 <= 5) print(5 >= 5) print(4 <= 5) print(6 >= 5)

Important: == versus =

Remember that == (double equals) is used for comparison and returns a boolean value. The single equals sign = is the assignment operator and assigns a value to a variable. Using = when you mean == is a common error.
